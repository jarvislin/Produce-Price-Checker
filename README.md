蔬果行情站 - Android
==========
<img src="https://lh6.ggpht.com/GCDvi_WtOavfDC074RsXM3JdABUsLjWLEbKVrNGxxm7XAQUb5d7oTg1a7qylOsOQ1Q=h900-rw" width="253" height="450">

透過農委會的農產品交易行情站取得蔬菜、水果的批發價格，

並根據設定顯示資料，提供使用者蔬果的參考價格。

設定中可以選擇批發市場、重量單位，

並可以勾選消費者模式，檢視包含利潤的價格區間，

無論是批發蔬果或是一般消費者皆可以使用。
